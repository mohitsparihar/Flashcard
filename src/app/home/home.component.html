Flashcard is a new app used for creating flashcards and learn through them. This app provides an easy to use interface for the learning and creating the data for flashcards. User can generate flashcards by folowing the three simple steps:-
  • Click on the create button from the navigation bar on the top. A form like this will appear.
  • Choose the topic and create your first flash card element.
  • For adding mutiple cards click on the add button then fill the data for the other card.
